# FreeCAD-MCP Universal Installer

An interactive, cross-platform Node.js CLI tool designed to automate the installation of the FreeCAD-MCP addon and seamlessly configure Model Context Protocol (MCP) environments for multiple AI clients including Google Gemini, Anthropic Claude, Goose, and Aider.

## Features

- **Multi-Client Support**: Out-of-the-box configuration files creation for Gemini CLI, Claude Desktop/Code, Goose CLI, and Aider CLI.
- **Cross-Platform**: Tailored path resolution and directory creation for macOS, Linux, and Windows systems.
- **Interactive Localization**: Fully dynamic interface built with localized support for English, Indonesian, Mandarin, and Thai.
- **Dependency Automation**: Validates system environments (Node.js 16+ and Git) and assists with missing dependencies natively.
- **Token Efficiency Toggle**: Includes dedicated options to initialize configurations in token-saving modes (`--only-text-feedback`).

## Prerequisites

Before executing the setup tool, please ensure your machine has:
- **Node.js** version 16 or higher
- **Git** installed on your system path
- At least one of the supported AI Clients already installed on your system

## Installation & Usage

1. Clone or download this installer repository to your workspace.
	```bash
	git clone https://github.com/cmalf/freecad-mcp-universal-installer.git
	```
2. Initialize the project or add the required configuration script.
	```bash
	cd freecad-mcp-universal-installer
	```
3. Launch the interactive CLI terminal via Node:

	```bash
	node fui.js
	```
4. Launch the interactive CLI terminal via Python:
	```bash
	python fui.py
	```

	or

	```bash
	python3 fui.py
	```

## Alternatively, if configured within your local project manifests(NodeJS):
	
```bash
npm start
```
	

Supported Configuration Footprints
----------------------------------

The tool handles automated file writes and payload injection for the following architecture:

### 1\. Google Gemini CLI

-   **Path Resolution**: `~/.gemini/settings.json` (Unix) | `%USERPROFILE%\.gemini\settings.json` (Windows)

-   **Format**: JSON structured with explicit `mcpServers` allowances and interactive shell toggles.

### 2\. Claude Desktop & Claude Code CLI

-   **Path Resolution**: Resolves dynamically to AppData (Windows), Application Support (macOS), or `.config` (Linux) schemas.

-   **Format**: Standardized JSON configuration blocks designating `uvx` executions.

### 3\. Goose CLI

-   **Path Resolution**: Dedicated platform paths targeting target `config.yaml`.

-   **Format**: YAML structure appending system extensions blocks directly.

### 4\. Aider CLI

-   **Path Resolution**: Local root project directory or user home path tracking `.aider.conf.yml`.

-   **Format**: YAML declarative parameters controlling `mcp-output` and argument array bindings.
